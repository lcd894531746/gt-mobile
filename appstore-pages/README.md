# GT Win App Store Pages

This folder contains static pages that can be used directly for App Store Connect:

- `index.html`: landing page
- `support.html`: support page
- `privacy.html`: privacy policy page

## Recommended usage

Upload the entire `appstore-pages` folder to any public static hosting service, such as:

- GitHub Pages
- Alibaba Cloud OSS static website hosting
- Tencent Cloud COS static website hosting
- Any public Nginx site

## Suggested App Store Connect fields

- Support URL: deployed `support.html`
- Privacy Policy URL: deployed `privacy.html`

## If you use GitHub Pages

Example:

- `https://example.github.io/gt-mobile/appstore-pages/support.html`
- `https://example.github.io/gt-mobile/appstore-pages/privacy.html`

## Before publishing

- Confirm the support email is correct
- Confirm the phone number is suitable for public display
- Confirm the address is the intended public business address
- Confirm the privacy policy matches the app's actual data usage
